# QUICK START GUIDE

## Introduction
This guide will help you quickly start using the Sip-ALG-checker.

## Prerequisites
Before you begin, ensure you have the following:
- A compatible platform (Linux, MacOS, or Windows)
- Required dependencies installed

## Installation
To install Sip-ALG-checker, follow these simple steps:
1. Clone the repository:
   ```bash
   git clone https://github.com/gauthiervq-sys/Sip-ALG-checker.git
   ```
2. Navigate to the folder:
   ```bash
   cd Sip-ALG-checker
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

## Usage
To start using the Sip-ALG-checker, run the following command:
```bash
npm start
```
